# HWGA
> HERE WE GO AGAIN — коллекция практических задач по C# и .NET с решениями и тестами.

## Цель проекта
Собрать структурированную коллекцию учебных задач.

Каждая задача:
* имеет чёткое описание и примеры;
* сопровождается рабочим решением на C#;
* включает тесты для проверки корректности.

## Атрибуция и использование ИИ
- **Код проекта** (файлы `.cs`) написан автором и распространяется под лицензией MIT.
- **Описания задач** (файлы `README.md` в папках задач) частично сгенерированы с помощью ИИ‑инструментов. Автор редактирует и проверяет их корректность.
- **Сторонние библиотеки** (xUnit, NUnit, Entity Framework и т. д.) распространяются под собственными лицензиями. Ознакомьтесь с ними в документации NuGet‑пакетов.
- При использовании описаний задач в других проектах рекомендуется ссылаться на этот репозиторий.
