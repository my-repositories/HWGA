# HWGA
> HERE WE GO AGAIN — коллекция практических задач по C# и .NET с решениями и тестами.

## Цель проекта
Собрать структурированную коллекцию учебных задач.

Каждая задача:
* имеет чёткое описание и примеры;
* сопровождается рабочим решением на C#;
* включает тесты для проверки корректности.

**Уровни сложности:**

## Как работать с проектом
1. **Выберите тему** из папки `src/app` (например, `Theme01.CLR`).
2. **Откройте уровень сложности** (`Easy`, `Medium` или `Hard`).
3. **Найдите задачу** — папка с названием задачи содержит:
   - `README.md` — описание задачи;
   - `.cs`-файл — решение (если готово);
   - `tests/` — тесты для проверки.
4. **Решите задачу** и запустите тесты.
5. **Повторите для других задач и тем.**

## Примеры задач по темам
- **Theme01.CLR:**
  - `EasyValueTypeVsReferenceType` — сравнение типов значений и ссылочных типов;
  - `MediumObjectLifetimeAndGC` — жизненный цикл объектов и сборка мусора;
  - `HardJitCompilationAndOptimization` — JIT-компиляция и оптимизация кода.
- **ThemeXX_EFCore:**
  - работа с контекстом базы данных;
  - настройка миграций;
  - запросы с использованием LINQ.

## Требования для разработки
