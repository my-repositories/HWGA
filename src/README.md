# Исходный код проекта

Этот раздел содержит исходные файлы проекта, организованные по темам и сложностям задач. Вся структура разработана с учетом лучшей практик для улучшения читаемости и поддерживаемости кода.

## Основная структура

```
src/
├── app/                  # Исходный код приложений и решений по темам
│   ├── Core/             # Общие компоненты и интерфейсы
│   │   ├── AssemblyTypeProvider .cs
│   │   ├── BaseProgram.cs
│   │   ├── IApp.cs
│   │   ├── IProgram.cs
│   │   ├── ITypeProvider.cs
│   ├── App.cs            # Основной класс приложения
│   ├── HWGA.csproj       # Проектный файл для приложения
│   ├── MainProgram.cs    # Класс с основной логикой программы
│   └── Темы (Theme00.Initial, Theme01.CLR, ...)
│       └── Задачи с префиксами Easy/Medium/Hard
├── tests/                # Тесты для задач
└── tools/                # Утилиты для автоматизации
    └── ReadmeUpdater.csproj # (Утилита для автоматического обновления README)
```

## Подробное описание

### `app/`
Директория содержит все исходные файлы и решения задач, организованные по темам.

- **`Core/`** — общие компоненты и интерфейсы, используемые различными темами.
  - `AssemblyTypeProvider.cs` — предоставляет информацию о типах сборок.
  - `BaseProgram.cs` — базовый класс для программных модулей.
  - `IApp.cs`, `IProgram.cs`, `ITypeProvider.cs` — интерфейсы для различных компонентов.

- **`App.cs`** — основной класс приложения, который инициализирует и запускает основные функции.

- **`HWGA.csproj`** — проектный файл для компиляции исходного кода приложения.

- **`MainProgram.cs`** — класс с основной логикой программы, который объединяет различные темы и задачи.

- **Темы (Theme00.Initial, Theme01.CLR, ...)** — директории для каждой темы, содержащие задачи с префиксами `Easy`, `Medium` и `Hard`.
  - Каждый файл внутри директорий соответствует конкретной задаче, например:
    ```
    src/
    └── app/
        └── Theme00.Initial/
            └── HardHelloWorld/
                ├── Impl/               # Реализации интерфейсов
                │   ├── HelloWorldFactory.cs
                │   ├── HelloWorldImplementation.cs
                │   ├── HelloWorldString.cs
                │   ├── HelloWorldStringImplementation.cs
                │   ├── PrintStrategyFactory.cs
                │   ├── PrintStrategyImplementation.cs
                │   ├── StatusCodeImplementation.cs
                │   └── StringFactory.cs
                └── Interfaces/         # Интерфейсы задачи
                    ├── IHelloWorld.cs
                    ├── IHelloWorldString.cs
                    ├── IPrintStrategy.cs
                    └── IStatusCode.cs
    ```

### `tests/`
Директория содержит тесты для исходного кода, организованные по темам и типам тестов.

- **`app.integration/`** — интеграционные тесты для приложений и решений по темам.
  - Пример файла: `HardHelloWorldIntegrationTests.cs`

- **`app.unit/`** — юнит-тесты для приложений и решений по темам.
  - Пример файла: `HelloWorldProgramTests.cs`

### `tools/`
Директория содержит утилиты для автоматизации различных задач, таких как обновление README.

- **`ReadmeUpdater.csproj`** — проектный файл для инструмента обновления README.
