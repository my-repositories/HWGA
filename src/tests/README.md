# Тесты проекта

Этот раздел содержит тесты для исходного кода проекта, организованные по типам тестов (интеграционные и юнит-тесты) и темам. Вся структура разработана с учетом лучших практик для улучшения надежности и поддерживаемости тестового кода.

## Основная структура

```
src/
└── tests/
    ├── app.integration/  # Интеграционные тесты для приложений
    │   └── Темы (Theme00.Initial, Theme01.CLR, ...)
    │       └── Задачи с префиксами Easy/Medium/Hard
    ├── app.unit/         # Юнит-тесты для приложений
    │   └── Темы (Theme00.Initial, Theme01.CLR, ...)
    │       └── Задачи с префиксами Easy/Medium/Hard
    └── tools.integration/  # Интеграционные тесты для утилит
        └── Сервисы и стратегии
    └── tools.unit/         # Юнит-тесты для утилит
        └── Сервисы и стратегии
```

## Подробное описание

### `app.integration/`
Директория содержит интеграционные тесты для приложений, организованные по темам.

- **Темы (Theme00.Initial, Theme01.CLR, ...)** — директории для каждой темы, содержащие тесты задач с префиксами `Easy`, `Medium` и `Hard`.
  - Пример файла: `HardHelloWorldIntegrationTests.cs`

### `app.unit/`
Директория содержит юнит-тесты для приложений, организованные по темам.

- **Темы (Theme00.Initial, Theme01.CLR, ...)** — директории для каждой темы, содержащие тесты задач с префиксами `Easy`, `Medium` и `Hard`.
  - Пример файла: `HelloWorldProgramTests.cs`

### `tools.integration/`
Директория содержит интеграционные тесты для утилит.

- **Сервисы и стратегии** — директории для различных сервисов и стратегий, содержащие интеграционные тесты.
  - Пример файла: `FileServiceIntegrationTests.cs`

### `tools.unit/`
Директория содержит юнит-тесты для утилит.

- **Сервисы и стратегии** — директории для различных сервисов и стратегий, содержащие юнит-тесты.
  - Пример файла: `FileServiceTests.cs`

## Используемые технологии
Тесты написаны с использованием фреймворков:
- **xUnit**

Эти инструменты обеспечивают надежность и точность проверок, что позволяет эффективно обнаруживать и исправлять ошибки в коде.
