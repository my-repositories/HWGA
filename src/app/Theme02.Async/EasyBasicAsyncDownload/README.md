## BasicAsyncDownload (Easy)

**Задача:** Реализовать асинхронное скачивание содержимого веб‑страницы по URL. Метод должен возвращать текст HTML‑страницы в виде строки.

**Пример:**

Вход: `"https://example.com"`  
Выход: `"<!DOCTYPE html>...<title>Example Domain</title>..."`  
Объяснение: Метод асинхронно загружает HTML‑код страницы и возвращает его в виде строки.

Вход: `"https://httpbin.org/delay/2"`  
Выход: `"..."` (содержимое страницы после 2‑секундной задержки)  
Объяснение: Даже при задержке на стороне сервера метод не блокирует поток и корректно возвращает данные.

**Ограничения:**
* Использовать `HttpClient` и ключевые слова `async`/`await`.
* Метод должен иметь сигнатуру `public async Task<string> DownloadPageAsync(string url)`.
* Обрабатывать исключения (например, `HttpRequestException`) и возвращать пустую строку в случае ошибки.
* Время выполнения для корректного URL не должно превышать 10 секунд.

**Что проверить в решении:**
* Корректность возврата HTML‑содержимого для валидных URL.
* Возврат пустой строки при сетевых ошибках или недоступности ресурса.
* Отсутствие блокировки основного потока во время выполнения HTTP‑запроса.

**Решение:** Смотрите `Program.cs`.
