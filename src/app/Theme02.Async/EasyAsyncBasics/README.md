## AsyncBasics (Easy)
Задача: Реализовать асинхронный метод, который имитирует скачивание данных из нескольких источников параллельно и возвращает общую сумму длин полученных строк.
Пример:
Вход: string[] urls = { "source1", "source2" }
Выход: 14
Объяснение: Каждое "скачивание" занимает 100мс. "source1" (7 симв.) + "source2" (7 симв.) = 14.
Ограничения:

* Использовать Task.Delay для имитации задержки сети;
* Метод должен возвращать Task<int>;
* Задачи должны выполняться одновременно, а не последовательно.

Что проверить в решении:

* Использование ключевых слов async и await;
* Применение Task.WhenAll для эффективного ожидания;
* Корректность подсчета суммы после завершения всех задач.
