## ValueTypeVsReferenceType (Easy)
Задача: Реализовать метод, который демонстрирует различие в копировании данных для struct и class. Нужно изменить поле в копии объекта и показать, как это влияет (или не влияет) на оригинал.
Пример:
Вход: Создание struct PersonStruct и class PersonClass с полем Age = 20.
Выход: Изменение Age у копий. Результат: у структуры оригинал 20, у класса оригинал 25.
Объяснение: Структуры копируются по значению, а классы — по ссылке.
Ограничения:

* Использовать только базовые типы C#;
* Запрещено использование ключевого слова ref в параметрах методов.

Что проверить в решении:

* Корректное объявление struct и class;
* Понимание того, что изменение копии структуры не меняет оригинал;
* Понимание того, что изменение по ссылке меняет состояние объекта в куче.
