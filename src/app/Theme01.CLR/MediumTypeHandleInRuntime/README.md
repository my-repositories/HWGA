## TypeHandleInRuntime (Medium)
Задача: Написать метод, который принимает массив объектов object[] и с помощью механизма рефлексии (RuntimeTypeHandle) группирует их по типам, подсчитывая количество экземпляров каждого типа.
Пример:
Вход: new object[] { "hi", 10, "bye", 20, 30 }
Выход: String: 2, Int32: 3
Объяснение: Метод определил типы объектов в рантайме и агрегировал статистику.
Ограничения:

* Не использовать метод GetType(), работать через Type.GetTypeHandle;
* Обработать случай, если массив содержит null.

Подсказка: Для получения типа через дескриптор используйте статический метод Type.GetTypeFromHandle(). Шаблон: namespace HWGA.CLR.Medium.TypeHandleInRuntime;
Что проверить в решении:

* Правильное извлечение метаданных типа из CLR;
* Корректная обработка null значений (пропуск или отдельная категория);
* Эффективность группировки (например, использование Dictionary).

Решение: Смотрите Program.cs.
